注册操作步骤

 1、进入商富贷首页，点击左上角"注册"按钮。
2、此时您已进入注册页面，根据提示信息，填写您的注册邮箱、用户名、密码，角色，点击 "就完成了。
3、注册成功后，系统会发送一封账户激活邮件到您的注册邮箱，进入注册邮箱，点击激活链接，激活账户。

注册时，一直没有收到确认电子邮件怎么办？

 1、请先到您的垃圾邮件文件夹里面寻找确认邮件
2、个别邮箱会误屏蔽商富贷确认邮件。如果你也碰到类似问题，请尝试使用由另一个邮件服务商提供的邮箱重新收取确认邮件。强烈建议你使用gmail免费邮箱、网易163免费邮箱、126免费邮箱，QQ邮箱。可随时联系网站客服人员。

免费注册会员

填写资料操作步骤

 1、登录系统成功后，进入我的账户
2、在基本设置那里填写个人资料，个人资料包括：房产资料、单位资料、私营业主、财务状况、联系方式、配偶资料、教育背景等
3、资料完善的好处：资料上传的越详细完整，获得的信用积分相应也会增加

填写资料

实名认证操作步骤

 1、登录系统成功后，进入我的账户
2、在认证中心菜单下面，进入实名认证
3、填写姓名，身份证号码，出生年月，户籍所在地，同时上传身份证正反面
4、注：实名认证一旦通过后将不能再修改真实姓名，请谨慎操作

实名认证

关于VIP的权利和义务

 为了商富贷网站的稳定运营，方便更多用户参与网络借款中介平台，投资者成为VIP会员不需要提交资料，VIP会员享有坏账保障的功能。
　　1.用户先在网站上注册账户，并向其账户充值150元，提交VIP申请；等审核通过后，网站自动扣除VIP费用。
　　2.借入者成为VIP后，一年以内可在信用额度内多次借款，并享有客服跟踪服务。
　　3.借出者申请VIP时需提交身份证及联系电话，提现银行账号户名与身份证一致。
　　4.借出者成为VIP后，借出本金享有商富贷网站合作商的全额担保，最大限度降低投资风险。
　　5.VIP用户同时享有借入和借出权利，商富贷网站将根据个人资料及信用积分审核信用借款额度，并根据还款情况及借出资金总额作出调整。

申请VIP

上传资料操作步骤

 1、登录系统成功后，进入我的账户
2、在认证中心菜单下面，进入资料上传页面，资料类别主要有：居住证(暂住证)、住房公积金、社保、行驶证、驾驶证等等
3、资料上传的好处：资料上传的越详细完整，获得的信用积分相应也会增加

上传资料

视频认证操作步骤

 1、登录系统成功后，进入我的账户
2、在认证中心菜单下面，进入视频认证页面，点击申请视频认证，然后联系客户，进行安排视频认证，客服在于您视频完毕后会进行认证通过操作
3、视频认证能增加您的信用积分

视频认证

申请操作步骤

 1、登录系统成功后，进入我的账户
2、在个人账户主页面上，点击"额度申请"，进入额度申请页面，提交自己需要申请的信用借款额度，交由审核部门审核，审核人员会根据您的自身信用状况给予一定的信用借款额度


申请额度

 目前系统的借款标种有：秒还标，信用标，给力标，净值标，担保标，用户可根据自身情况选择发布不同类型借款标。
